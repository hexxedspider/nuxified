Quick Setup

Get started in minutes

1

Clone Repository

git clone https://github.com/hexxedspider/nuxified.git
2

Install Dependencies

pip install -r requirements.txt

Recommended: Python 3.11.9 or compatible version

3

Configure Environment

Rename .envv to .env and add your Discord token

Optional: Add API keys for OpenRouter (AI), Steam, Spotify, etc.

4

Launch

python nuxified.py

Or use the GUI launcher: python launcher.py
